
# Responsive Fried Chicken Website

## Project Overview

This project is a fully responsive restaurant website for a fried chicken brand, crafted entirely with *HTML*, *CSS*, and *JavaScript*. It stands as a comprehensive front-end development project that demonstrates proficiency in building modern, visually rich, and highly functional websites without the use of any external frameworks or libraries. Every aspect of the site — from layout structure to interactive behaviour — was designed and implemented from scratch, reflecting a strong command of core web technologies.

---

## Approach

The website was built with a deliberate focus on *user experience and visual quality*. The design language is clean, contemporary, and appetising — appropriate for a food brand that needs to attract and retain the attention of visitors from the moment they land on the page. Every section was carefully composed to guide the user naturally through the content, balancing aesthetics with functionality.

A central part of the design process was adopting the *Mobile First methodology* a development approach that prioritises designing for the smallest screen sizes before scaling up to larger viewports. This is considered industry best practice in modern web development, as it ensures the site's core content and functionality are accessible to mobile users first, with progressive enhancements layered on top for tablet and desktop experiences. The result is a layout that feels native and comfortable on any device — from a compact smartphone screen to a widescreen desktop monitor.

---

## Features & Functionality

### Scroll-Triggered Animations
The website features **carefully timed animations that activate as the user scrolls** through the page. Rather than presenting all content statically, elements enter the viewport with fluid motion — adding depth, elegance, and a sense of interactivity to the browsing experience. These animations were powered by **ScrollReveal.js**, combined with CSS transitions and keyframe animations, striking a balance between visual richness and page performance.

### Smooth Scrolling Navigation
Navigation between sections is handled with **smooth scrolling behaviour**, replacing the jarring, instant jumps typical of standard anchor links. This creates a polished, app-like feel that significantly elevates the perceived quality of the website. Users can effortlessly glide between sections, making the experience feel cohesive and professionally crafted.

### Slider
The website incorporates a **Swiper.js-powered image carousel**, enabling smooth, touch-friendly sliding of featured content. This adds an interactive, modern dimension to the layout and is fully responsive across all devices.

### Full Cross-Device Compatibility
The website has been thoroughly tested and optimised for **compatibility across all modern mobile devices and screen resolutions**. Using fluid grid layouts, flexible images, and carefully constructed CSS media queries, the site adapts gracefully to any viewport — ensuring that no user, regardless of their device, encounters a broken or degraded experience.
